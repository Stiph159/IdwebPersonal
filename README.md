# Proyecto Web Personal

URL del proyecto hospedado:
https://stiph159.github.io/IdwebPersonal/

Descripción:
Este proyecto consiste en una página web desarrollada de manera individual como trabajo final del curso Introducción al Desarrollo Web, la cual presenta una estructura organizada y un diseño básico que integra estilos visuales y elementos interactivos, permitiendo reforzar los conceptos fundamentales del desarrollo web mediante el uso correcto de HTML, CSS y JavaScript.

Lenguajes utilizados:
- HTML
- CSS
- JavaScript
